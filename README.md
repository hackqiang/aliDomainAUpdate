# aliDomainAUpdate
##update the A record for Ali Domains

add your own accesskeys in `config/key.json`:

	{

    	"access_key_id": "XXXXX",

    	"access_key_secret": "XXXX"

	}

add your own domains in `config/domains.json`:

	[
    	"hackqiang.org"
	]



`aliDomainAUpdate` used for ubuntu service
